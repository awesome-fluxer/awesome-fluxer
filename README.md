## Libraries

- [Fluxer.py](https://github.com/akarealemil/fluxer.py) (Python) - A Python API Wrapper for Fluxer App. 
- [Fluxify](https://github.com/Eschryn/Fluxify) (C#) - A .NET library for Fluxer.
- [Fluxer.NET](https://fluxer.net/) (C#) - A .NET library for building bots and applications on the Fluxer platform.
- [Fluxer.JS](https://fluxer.js.org/) (JS) - An SDK for building bots on Fluxer.
- [FluxerGo](https://github.com/fluxergo/fluxergo) (Go) - A Go Library for building Fluxer applications.
- [fluxer-rust](https://github.com/vesaber/fluxer-rust) (Rust) - A Rust written API Wrapper for the Discord alternative Fluxer.
- [Fluxer.RUST](https://github.com/DeviMorris/Fluxer.RUST) (Rust) - Full Rust API library for Fluxer.

## Bots

- [Fluxy](https://fluxy.dorcus.digital/) - Everything you need to run a safe, well-organized Fluxer server.
- [Bloo](https://bloo-bot.xyz/) - A powerful, bubbly Fluxer bot packed with moderation, XP ranking, webhooks, tickets, and so much more — all wrapped in one slimy little package.
- [Nevi](https://nevi.nev.so/) - A powerful, feature-rich Fluxer bot built for communities that demand more.
- [Fishing Bucket](https://github.com/HellishBro/FishingBucket) - Fishing Bucket is a Fluxer bot that proxies messages as different users.
- [Fluxer.Red](https://github.com/Red-Fluxer-Patches/Red-DiscordBot) - A multi-function Discord bot patched to work with Fluxer (COMPLETELY UNSUPPORTED).

## Tools

- [fluxer-rpc](https://github.com/letruxux/fluxer-rpc) - mirror your discord rpc to fluxer! 〰️
- [Midnight Tools](https://github.com/BubbaXM/Midnight-Tools/) - Midnight is a custom add-on experience for Fluxer desktop, similar in spirit to Vencord-style customization for Discord clients.
- [BetterFluxer](https://github.com/RoxyBoxxy/BetterFluxer) - Plugin Injector for the Fluxer.app desktop client
- [Reflux](https://github.com/Its3rr0rsWRLD/Reflux) - A simple plugin injector for Fluxer.

## Bridges

- [Fluxcord](https://fluxcord.jbcrn.dev/) - A Discord <-> Fluxer bridge. 
- [Bifröst](https://github.com/KartoffelChipss/bifrost) - A Discord-Fluxer-Bridge
